//バリデーション
　ログイン
　  name
　　　必須、max50字、文字形式
　  エラーメッセージ
　  　必須：名前を入力してください
　  　max50：50字以内で入力してください
　  　文字形式：文字形式で入力してください

　  password
　　　必須、min8、max20、数字必須、アルファベット必須
　  エラーメッセージ
　  　必須：英数字込みでパスワードを入力してください
　  　min8：8字以上で入力してください
　  　max20：20字以内で入力してください
　  　数字必須：数字を含めてください
　  　アルファベット必須：アルバイトを含めてください

　新規登録
    name
    email
    password

    rating
    comment

　レビュー登録
    title
    authore

    image_url
    genre

    name
