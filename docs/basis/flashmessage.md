## scusses
    web

    書籍更新：書籍を更新しました
    書籍削除：書籍を削除しました
    レビュー登録：レビューを登録しました
    レビュー更新：レビューを更新しました
    レビュー削除：レビューを削除しました
    ジャンル登録：ジャンルを登録しました
    ジャンル更新：ジャンルを更新しました
    ジャンル削除：ジャンルを削除しました

    応用
    読書計画登録：読書計画を作成しました
    読書計画更新：読書計画を更新しました
    読書計画削除：読書計画を削除しました
    complete：この書籍のステータスを読了に変更しました
        
    API　ステータスコード
    書籍一覧取得：書籍一覧の取得に成功しました　200
    書籍詳細取得：書籍の詳細取得に成功しました　200
    書籍登録：書籍の登録に成功しました　201
    書籍更新：書籍の更新に成功しました　200
    書籍削除：204　//エラーメッセージなし

## error
    ジャンル削除：紐づいている書籍があるため削除できません

    API　ステータスコード
    詳細、更新、削除(存在しないID指定)：書籍が存在しません　404
    更新、削除（認可エラー）：この操作を行う権限がありません　403
    未認証時：ログインしてください　401
