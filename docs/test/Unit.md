単体テスト
・バリデーションテスト
　必須項目未入力、最大値超過、一意性の担保

・正常系テスト
　フラッシュメッセージが表示されるか
　ページネーションで取得できているか
　バリデーションのエラーメッセージが表示されるか
　期待するフォームが画面に表示されているか
　ランキングが降順に表示されているか
　検索機能が動作するか（API）
　ジャンル絞り込みが動作するか（API）
　
・異常系テスト
　存在しないIDを渡すとエラーが返るか
　ログインしていない状態でアクセスするとリダイレクトされるか
　他人のデータを更新・削除しようとするとリダイレクトされるか

・境界値テスト
　バリデーションのmax値を超えたらバリデーションが働くか
　max値ちょうどで登録できるか
